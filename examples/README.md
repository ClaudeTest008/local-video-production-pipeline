# Examples

| File | What it is |
|---|---|
| `comfyui-txt2img.json` | Minimal SDXL text-to-image workflow (ComfyUI API format). Queue it via `POST /api/comfyui/queue` or paste it into the ComfyUI page's queue box. |
| `pipeline-walkthrough.md` | End-to-end idea → video walkthrough using the API. |

Seed data: `cd backend && python seed.py` creates a sample project ("Vertical Farming Deep-Dive") with a script, storyboard scenes, prompts, an SEO pack, all 14 agent profiles, and the full MCP catalog.
